# Sales Report Dashboard

## Project Overview

This project presents an interactive Power BI dashboard developed using sales data stored in Excel. The dashboard provides insights into revenue performance, product sales, regional distribution, and salesperson contributions.

The objective of this project is to transform sales data into actionable business insights that support strategic decision-making and performance monitoring.
